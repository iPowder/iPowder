The software requires Labview and Python3.10 (with numpy) operating environments.

Labview runtime environment can be down load in https://www.ni.com/zh-cn/support/downloads/software-products/download.labview-runtime.html#521521

As for python, you can install python3.10 on your computer and put it in an environment variable, or use anaconda to create an environment called py310. After that please remember to install the numpy library for python310.

If you want to use the search-match, please download the Crystal_db databases in https://workdrive.zohopublic.com.cn/folder/bgee25b186e54186b497097286605796b23be?layout=list or https://github.com/iPowder/Crystal_db. And make sure that Crystal_a.db,Crystal_b.db,Crystal_c.db,Crystal_d.db are in folder like /iPowder/Crystal_db/ , where /iPowder is the directory where the iPowder.exe program resides.
